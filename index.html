¿Cómo funciona?

ColorFly Studio te permite generar paletas de colores de forma rápida y sencilla. Selecciona la cantidad de colores y el formato de código que prefieras (HEX o HSL), genera una nueva paleta y guarda tus combinaciones favoritas para utilizarlas en cualquier proyecto de diseño o desarrollo web.
